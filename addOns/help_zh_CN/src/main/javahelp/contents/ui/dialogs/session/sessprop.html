URL正则表达式

在 从*中排除 对话框，您可以输入正则表达式来定义排除的URL。虽然您可以用反斜线转义单个元字符，您也可以使用 \Q...\E 转义序列。中的所有字符 \Q 和 \E 被解释为原义字符。例如:\Q*\d+*\E匹配文本*\d+*。当您通过一些上下文菜单排除URL时，此转义序列用于ZAP。
备注: 如果您的URL包含一个"\E"，那么当使用时您必须按照以下步骤做 \Q...\E 转义序列:
  • 打开转义序列
  • 在"字符"\E之前关闭转义序列
  • 转义反斜线
  • 在"\E"之后打开另一个转义序列；
  • 按正常方式关闭转义序列。

Example: subdomain.example.com/path?a=\E&moredata=2 should appear as \Qsubdomain.example.com/path?a=\E\\E\Q&moredata=2\E
