附加组件

附加组件是ZAP的附加功能
它们可以完全访问所有的ZAP内部，因此可以提供非常强大的新功能。
您可以在在线附加组件市场动态添加安装附加组件通过 Manage Add-ons dialog.
您通常可以在ZAP UI中添加和删除附件，而无需重新启动它。

Add-ons can provide context sensitive help which is dynamically integrated with the help file.
If you are reading this page via a ZAP help file (as opposed to reading it online) then any help pages associated with the add-ons you have installed will be available under here.

附加组件会被分配一个状态，可能是下列之一：
     发布版which indicates they are of high status and fit for purpose
     测试版which indicates they are of reasonable status but may be incomplete or need further testing
     Alpha版这表明它们正处于发展的早期阶段。

可用于ZAP的附加组件的位置

要为ZAP提供附加组件，它必须位于以下位置之一：
  • plugin目录位于ZAP的安装文件夹中;
  • plugin 目录位于ZAP的用户文件夹中。
