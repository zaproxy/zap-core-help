Mga direktoryo

Isang listahan ng mga direktoryo mula sa kung saan ang mga script ay mai-load. Babasahin (at isulat) ang ZAP ang mga script gamit ang encoding ng character na UTF-8.

Ang iskrip ay dapat nasa subdirectories na pinangalanang matapos ang may-katuturang script uri (tulad ng 'aktibong', 'passive', 'proxy' atbp) at dapat magkaroon ng isang angkop na extension para sa ang wika ng script na ginamit.

Ang mga script na hindi maaaring isulat sa ay idadagdag sa seksyon ng Mga Template.

Ang repository ng script ng komunidad ng ZAP ay isang mahusay na pagpipilian upang idagdag: https://github.com/zaproxy/community-scripts

Kakailanganin mong i-clone ang isang lokal na kopya ngunit maaari mong ma-access ang lahat ng mga script na idagdag ng komunidad, at kahit na mag-upload ng iyong sariling mga script sa pamamagitan ng mga kahilingan ng pull GitHub.
