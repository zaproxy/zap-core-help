Ang pagsisimula

The quickest way to get going with ZAP is to use the Quick Start add-on, which is installed by default.
This allows you to enter a URL which ZAP will first spider and then active scan.
For a more in depth test you should explore your application using your browser or automated regression tests while proxying through ZAP.

At its heart ZAP is a manipulator-in-the-middle proxy.
To get the most out of ZAP you need to configure your browser or functional tests to connect to the web application you wish to test through ZAP.
If required you can also configure ZAP to connect through another proxy - this is often necessary in a corporate environment.

Ikaw ay maaaring magmadali at madaliang ilunsad ang mga browser na para maikumpigura sa proxy ng ZAP sa pamamagitan ng madaliang pagsisimula ng tab. Maaaring ito ay pwedeng mailunsad ng alinman sa mga pinaka karaniwang mga browser na na-install na may bagong mga profile.
kung nais mong gamitin ang alinman sa iyong browser na may umiiral nang profile, para sa halimbawa sa ibang browser ng mga add-on na naka-install, pagkatapos ay kinakailngan mo na ikumpigura ng manu-mano ang iyong browser para sa pag proxy sa pamamagian ng ZAp.
kung alam mo kung paano isaayos ang mga proxy sa iyong web na browser at oagkatapos ay magpatuloy at bigyan ito ng isang go!
kung ikaw ay hindi sigurado pagkatapos ay tingnan sa mga nakumpigurang mga proxy ng seksyon.

Kapag nakumpigura mo ang ZAP bilang iyong proxy na browser pagkatapos ay subukan na ikonekta sa mga web ng aplikasyon na iyong sinusubukan.
kung hindi mo makonekta dito saka mo siyasatin ang mga setting ng proxy muli. Kinakailngan mong masiyasat ang iyong mga browser na mga setting ng proxy, at ganun din ang proxy ng ZAP sa mga setting.
Ito ay mahalaga din sa pagsisiyasat na ang aplikasyon na iyong sinusubukan mong siyasatin ay tumatakbo!

Kapag ikaw ay matagumpay na nakakonekta sa iyong aplikasyon sa pammagitan ng iyong browser pagkatapos ay tumingin sa ZAP muli. Dapat mo ngayong makita ang isa o higit pang mga linya sa mga sayt at kasaysayan ng mga tab.
kapag tayo ay nasa negosyo. Kung hindi saka mo kinakailngan siyasatin ang iyong setting sa proxy muli.
