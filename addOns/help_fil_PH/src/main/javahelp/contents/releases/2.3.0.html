Ang sumusunod na mga pagbabago ay ginawa sa inilabas na ito:

Ang mga Makabuluhang pagbabago:

Ang Bersyon ng 'lite' sa ZAP

Para sa inilabas na ito kami ay nagbibigay ng isang 'lite' na bersyon ng ZAP sa karagdagan ng mga 'lubos' na bersyon. Naglalaman ito ng eksaktong magkaparehas sa core ng code, ngunit ito ay pinagssama lamang ng mas kaunting mga default add-on. Siyempre, maaari mong i-download ang lahat ng 'nawawalang' mga add-on mula sa pamilihan ng ZAP para sa 'pagpapataas' ng bersyon ng lite para sa isang lubos.
Ang 'lite' ang layunin ng isang bersyon sa seguridad ng mga bagong tao na nangangailangan ng mas kaunting paunang pag-andar na inaasahang magiging mas madali para magsimula. Ito rin ay nababagay para sa mga taong naghahanap ng isang mas kaunting pag-download o sa mga humihiling eksaktong ipasadya ang mga add-on na na-install nila.

Ang mga pag suporta para sa kaganapan ng Clien side ng (browser)

Ngayon maaari monagn tingnan, maharang, manipulahin at ipadalang muli at ang fuzz ay mga kaganapan sa client-side. Kinakailngan nito ng mga postMessage, kaya ito ay maaari mo ng subukan ang DOM batay sa XSS na mga kahinaan ng mga postMessage. Ito ay ang unang bahagi ng isang hanay ng mga planadong pagbabago na suportahan ang pagsusuri ng mga AJAX at HTML5 na aplikasyon kahit na epektibo.

Ang pagpapahusay ng pagpapatunay ng suporta

Ang pagpaptunay ng pag suporata ng ZAP ay inayos ng kumpleto para sa madaling pamamahala ng mga kumplikadong klase ng ng mga nag papatunay na paraan at mmga senaryo. ang pagsuporta ay idinagdag din para sa mga itinakdang mga iskrip na nagbibigay sa iyo nahawakan ang mga pasadyang pamaraan ng pagpapatunay. sa karagdagan, ang mga pag-unawa ngayon ng ZAP at pagpapahintulot na makumpigura ng mga gumagamit ng mga aplikasyon ng web kaya ang mga ibat-ibang aksyon ng lahat ng parte ng ZAP na maaring magsagawa mula sa punto ng mga ipinapakita ng mga nilinaw ng mga gumamit. para makapag-umpisa, suriin ang mga bagong Mga pagpapatunay at Ang mga Gumagamit na mga panel sa mga Sesyon ng mga katangian para sa bawat isa sa mga itinukoy na konteksto.

Ang pag-suporta para sa pamantayan ng mga app

Ang inilabas na ito ay kabilang ng pag suporta para sa mga 'solong pahina'ng mga aplikasyon at di-pamantayang key-value ng mga pang hiwalay. Maari mo nang kontrolin ang mga setting na ito sa pamamagitan ng bagong Istraktura sa mga Katangian ng panel ng Sesyon.

Ang mga bagong Input ng Vector kabilang ang mga iskripto ng user-defined

Ang mga pag suporta ng ZAP sa mga bagong opsyon para sa pagtukoy ng ang mga input ng vector ang mga kahilingan ng mga elemento ng ZAP sa pag attack. Ang mga bagong opsyon ay magagamit sa mga Ang mga aktibong iskan ng Input ng Vector ang mga panel ng opsyon. Ang supota na maaaring maidagdag para sa pagtukoy ng mga pasadyang iskrip na nagtutukoy ng bagong input ng mga vector.

Ang patakaran ng pag-iskan - ang mabuting grained ng kontrol

Ang patakaran ng pag-scan ngayon ay may isang mabuting grained na kontrol, Ang pagpapahintulot sa iyo upang baguhin ang indibidwal na mga patakaran ng scanner. Maari mo ring tukuyin, ang pag-load at pag-save ng mga patakaran ng pag-iskan, nagpapahintulot sa iyo na mapanatili ang isang hanay ng mga patakaran na gumawa ng mahusay sa iba't-ibang mga kalagayan.
Sa kagdagan, sa pamamagitan ng default ng ZAP ay hindi na ngayon ma-iskan ang mga kilalang parametro ng serbisyo (e.g. __VIEWSTATE) nagpapabilis ng kabuuang proseso ng pag-iskan. Ito ay kumpletong pag gamit ng pagkukumpigura, na nagpapahintulot sa iyo na eksaktong matukoy ang mga parametro na dapat balewalain ng ZAP.

Ang advance na diyalogo ng Aktibong Pag-iskan

Ang bagong Ang advance ng Aktibong Pag-iskan' na diyalogo ay nagpapahintulot sa iyo na eksaktong matuloy kung paano mo gustuhing paganahin ang aktibong scanner. Ito ay nagpapahintulot sa iyo ng pagtukoy ng mga ‘custom vector’ na pinagtitiwalaan na tukuyin ang anumang mga string na gusto mong atakihin. Ito rin ay sumusuporta ng opsyon na mag-scan ng anumang natukoy na mga gumagamit na maaari mong tukuyin para sa mga aplikasyon sa ilalim ng pagsasanay. Ang pag-uumpisa ng isang Aktibong iskan na advance sa pamamagitan ng Mga kasangkapan ng menu o sa pamamagitan ng Pag-atake seksyon ng right click popup na menu.

Ang mga opsyon ng pagpapalawak ng command line

Maaari mo ngayon na patakbuhin ang ZAP ng 'inline' halimbawa. ng walang pagsisimula ng ZAP UI o ng isang daemon. Sa kalakarang ito maaari mong patakbuhin ang mga simpleng atake o magpatakbo ng mga iskrip na maaaring mag-access sa lahat ng pagpapatakbo ng ZAP. Maari mo ngayong i-override ang kahit alinmang opsyon na tinukoy sa file ng kumpigurasyon sa pamamagitan ng command line ng mga parametro.

Ang lalo pang pagsuporta ng API

Ang pagpapahaba ng API upang suportahan ang mas higit pang pagpapatakbo ng ZAP.

Ang pang-international na tulong ng payl

Ang pang-international na tulong ng payl at ang nasa proseso ng katauhan ng pagsasalin ng maraming mga salita sa pamamagitan ng https://crowdin.com/project/owasp-zap-help. Kung ang pag gamit mo ng ZAP sa isa ng mga maraming salita na sinuportahan namin, pagkatapos ang mga tulong ng file ay kabilang ang lahat ng paggamit ng pagsasalin para sa wikang iyon habang binabalik ang default sa Ingles para sa mga kataga na hindi pa naisalin.

  • Ang Pranses

  • Ang Espanyol

Ang mga shortcut ng keyboard

Ang lahat ng aytem sa menu ay maaring nang pakiusapan sa pamamagitan ng keyboard na shortcut. Tinukoy ang mga default para sa katunayan ng lahat ng mga kaso, subalit maaring mo itong makumpigura sa iyong sariling kagustuhan sa mga Keyboard ng panel ng mga opsyon.

Ang bagong mga opsyon ng UI

Mayroon isang bagong opsyon sa pagpapabago ng display upang ang napiling tab ay tumatagal ng hanggang sa buong iskrin. kapag ginamit ang ZAP ay kapakipakinabang ito sa maliit na iskrin. ito ay mayroon ding isang opsyon na mag-toggle ng pagtanaw ng mga pangalan ng tab sa isang off para lalo pang makatipid ng espasyo.

Ang higit sa mga listahan ng UI ay napalitan na rin ng mga tables, kung saan pinahihintulutan ka na palitan ang lapad ng hanay at tukuyin ang eksakto ang anumang hanay na ipinapakita, at paano ang mga talahanayan ay pinagsunodsunod.

Ang karagdagang pagpapatakbo ng paglipat ng mga add-on

Karamihan sa mga pangunahing pagpapatakbo ay nailipat sa mga add-on kung saan ay nagpapahintulot sa atin na maghatid ng mga update sa pamamagitan ng ZAP ng pamilihan sa halip na nangangailangan ng mga bagong buong mga inilabas.
This includes the language packs, so translations made to the ZAP UI via https://crowdin.com/project/owasp-zap can be downloaded within ZAP or even automatically installed.

Bago at mas pinaganda na aktibo at pasibong pag-iskan ng mga patakaran

Many of the release status active and passive scanning rules have been improved. There are new alpha and beta status rules and many rules have been promoted from alpha to beta and from beta to release status.
