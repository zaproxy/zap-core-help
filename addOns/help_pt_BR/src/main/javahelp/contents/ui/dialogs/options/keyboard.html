Tela de opções de teclado

Esta tela permite que você configure os atalhos de teclado para todos os menus do ZAP.

Clicando em qualquer uma das linhas da tabela, aparecerá uma caixa de diálogo que permite alterar os atalhos de teclado.

Dicas de 'Ação'

Clicar neste botão irá gerar uma página HTML que mostra todos os atalhos de teclado, classificados por 'ação'.
Este botão aparece quando a API está habilitada e o seu sistema operacional suporta abrir URLs num browser.

Dicas de 'Teclas'

Clicar neste botão irá gerar uma página HTML que mostra todos os atalhos de teclado, classificados por 'tecla'.
Este botão aparece quando a API está habilitada e o seu sistema operacional suporta abrir URLs num browser.

Reset

Clicar neste botão vai retornar todos os atalhos a seu estado default.
