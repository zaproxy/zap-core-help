Diálogo de reenvio

Esta caixa de diálogo permite que você envie novamente uma solicitação, depois de fazer quaisquer alterações.

Aba de Request

Isto mostra o cabeçalho de request e os dados, seja em um ou dois painéis, dependendo das opções escolhidas.

Um menu pull down de 'Método' permite que você alterne entre os métodos HTTP.
Observe que quando o método é alterado "para" um POST, então os parâmetros de URL são movidos para o corpo, e quando o método é alterado "de" um POST, então, todos os parâmetros no corpo são movidos para a URL.

Menus pull-down permitem que você selecione diferentes Visualizações do cabeçalho de request e do corpo.

  Visualização dividida para cabeçalho e corpo

Aqui se altera a exibição, para que painéis separados sejam usados para o cabeçalho e para o corpo.

  Visualização combinada para cabeçalho e corpo

Isso altera a exibição para que o cabeçalho e o corpo sejam mostrados em um só painel.

  Usar a sessão de rastreamento atual

Consulte o item de menu 'Habilitar rastreamento de sessão (Cookie)' no Menu Editar.

  Seguir os redirecionamentos

Se selecionado automaticamente segue qualquer redirecionamento enviado ao navegador.

  Abas de Request e Resposta lado-a-lado

Isso altera a exibição para que as abas de Request e Resposta estejam lado a lado.
Isto aumenta o volume de informação que pode ser exibida, mas significa que você não pode ver o Request e a Resposta ao mesmo tempo.

  Request exibido acima da Resposta

Isso altera a exibição para que o painel de Request seja exibido acima do painel de resposta.
Isto reduz o volume de informação que pode ser exibida, mas significa que você consegue ver o Request e a Resposta ao mesmo tempo.

  Painéis de Request e Resposta ao lado

Isso altera a exibição para que o painel de Request seja mostrado à esquerda do painel de resposta.
Isto reduz o volume de informação que pode ser exibida, mas significa que você consegue ver o Request e a Resposta ao mesmo tempo.

Aba de Resposta

Isto mostra os dados enviados para seu navegador em resposta à solicitação que você enviou, em um ou dois painéis, dependendo das opções escolhidas.

Menus pull-down permitem que você selecione diferentes Visualizações para o cabeçalho de resposta e o corpo.

Menu do botão direito

Clicando no botão direito do mouse fará aparecer um menu que permitirá a você:

Localizar...

Isso irá abrir o diálogo Localizar.

Codificar/decodificar...

Isso irá abrir o diálogo Codificar/Decodificar.
Se você tem algum texto realçado, ele será automaticamente incluído na caixa de diálogo.
