Gerenciador da Política de Varredura

This allows you to manage the scan policies that define the rules that are run when performing an active scan.
You can have as many scan policies as you like, and choose which one of them is run when you perform an active scan.

Adicionar

The Add button launches the Scan Policy dialog with all of the settings defaulted.

Modificar

The Modify button launches the Scan Policy dialog with all of the selected scan policy.

Remover

O botão Remover exclui a política de varredura selecionada depois de você confirmar sua decisão na caixa de aviso.

Importar

O botão de Importação permite importar uma diretiva de varredura criada por outra instância do ZAP.

Exportar

O botão Exportar permite exportar uma política de varredura para que você possa importá-la para outra instância do ZAP, inclusive em outra máquina.
