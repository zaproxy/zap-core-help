Visualizações

ZAP fornece um conjunto de visualizações permutáveis que permite a você exibir as solicitações e respostas de maneiras diferentes.
As seguintes visualizações estão incluídas por padrão:
    TextoOs dados no formato de texto
    Visão HexadecimalUma tabela mostrando a representação hexadecimal de todos os caracteres no cabeçalho e corpo.
Em telas editáveis, isto permite que você adicione caracteres de controle utilizando seus códigos hexadecimais.
Você não pode adicionar ou excluir caracteres neste modo de exibição - para fazer isso mude para um dos outros modos de exibição e em seguida volte para continuar no modo de exibição hexadecimal.
    ImagemA imagem - disponível apenas para corpos que contenham imagens
    TabelaUma tabela com uma linha para cada atributo e valor.
Em telas editáveis todos os valores ganharão codificação de URL ao serem enviados.
    Tabela (adv)Uma tabela com uma linha para cada atributo e valor mais funções adicionais.
Em telas editáveis não haverá codificação automática para URL no envio, mas isso pode ser feito manualmente através da função 'URLEncode'.
    Request grandeUma visualização em destaque, usada para impedir que a interface do usuário tente carregar um corpo de request muito grande.
Você precisará mudar para uma visualização diferente para exibir o conteúdo real.
O limiar para o modo de exibição pode ser alterado através da telaOpções de visualização
    Resposta GrandeUma visualização em destaque, usada para impedir que a interface do usuário tente carregar um corpo de request muito grande.
Você precisará mudar para uma visualização diferente para exibir o conteúdo real.
O limiar para o modo de exibição pode ser alterado através da telaOpções de visualização

Observe que Complementos podem inserir botões adicionais.

Usado em

    Aba de interrupções/pausas
    Aba de Request
    Aba de Resposta
    Request manual
    Diálogo de reenvio
