Tab params

Ini menunjukkan ringkasan dari bidang header parameter dan tanggapan yang digunakan oleh sebuah situs.

Situs dapat dipilih melalui alat bar atau Tab situs.

Untuk setiap parameter yang dapat anda lihat:
     Jenis - Cookie, BENTUK, URL, atau header
     Nama parameter (atau tanggapan header)
     Berapa kali yang telah digunakan
     Jumlah nilai yang unik
     Perubahan persentase, di mana 0 berarti hanya satu nilai yang telah digunakan dan 100 berarti semua nilai yang unik
     Bendera - termasuk bendera cookie dan anti csrf dan sesi
     Beberapa nilai - keseluruhan nilai mungkin tidak terlihat

Menu klik kanan

Mengklik kanan pada sebuah node akan memunculkan menu yang memungkinkan anda untuk:

Telusuri

Ini akan menampilkan semua contoh parameter yang dipilih di Tab Pencarian.

Tandai sebagai token anti CSRF

Ini akan menandai parameter sebagai Token anti CSRF.

Tidak menandai token anti CSRF

Ini akan menghapus tanda token anti CSRF dari parameter.

Tandai sebagai sesi token

Ini akan menandai parameter sebagai sesi token untuk situs saat ini dan akan memberitahukan Sesi HTTP alat yang sesuai.

Tidak menandai sesi token

Ini akan tidak menandai parameter sebagai sesi token untuk situs saat ini dan akan memberitahukan Sesi HTTP alat yang sesuai.
