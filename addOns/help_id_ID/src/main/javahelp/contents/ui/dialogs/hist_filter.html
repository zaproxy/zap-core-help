Sejarah Filter dialog

Dialog ini membolehkan Anda untuk membatasi permintaan yang ditampilkan dalam tab Sejarah.

Bidang

Dialog memiliki bidang-bidang berikut:

Metode

Metode HTTP untuk menyaring.

Kode

Kode Respon HTTP untuk menyaring.

Tag

The Tag untuk menyaring.

Peringatan

Jenis Peringatan untuk menyaring.

Catatan

Apakah adanya atau tidak adanya Catatan harus disaring pada.

URL inc regex

Daftar 'termasuk' pola regex (satu per baris) - jika ditentukan maka hanya Url yang cocok dengan pola yang akan disertakan.

URL regex Sunan Hotel Solo

Daftar 'Kecualikan' regex pola (satu per baris) - jika ditentukan maka setiap Url yang cocok dengan pola yang akan disembunyikan.
