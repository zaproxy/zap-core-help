Me-Reset ke Default Pabrik

'Me-Reset ke Default Pabrik' tombol reset semua pilihan untuk nilai yang ditetapkan ketika anda pertama kali berlari ZAP. Ini termasuk reeseting host dan port yang ZAP adalah mendengarkan dan API key.
Salah satu item konfigurasi yang tidak akan berubah adalah ZAP Root CA Sertifikat. Jika anda ingin mengubah ini, maka anda dapat melakukannya melalui Sertifikat SSL Dinamis layar.

     Tingkat atas Alat menu'Pilihan...' item menu
