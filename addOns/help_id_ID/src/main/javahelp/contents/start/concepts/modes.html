Modus

ZAP memiliki 'mode' yang dapat:

  • Aman - tidak ada operasi yang berpotensi berbahaya yang diizinkan
  • Dilindungi - anda hanya dapat melakukan (berpotensi) tindakan-tindakan berbahaya pada Url dalam ruang Lingkup
  • Standar - seperti pada rilis sebelumnya, anda dapat melakukan apapun
  • SERANGAN - node baru yang ada di ruang Lingkup aktif dipindai segera setelah mereka menemukan
Disarankan bahwa anda menggunakan mode Perlindungan untuk memastikan bahwa anda hanya menyerang situs yang anda maksud.

Mode dapat diubah melalui toolbar (atau ZAP API) dan bertahan diantara sesi.

Contoh dari hal-hal yang tidak akan mungkin dilakukan dalam mode Aman atau dalam mode Perlindungan ketika tidak bertindak berdasarkan URL dalam Lingkupan:
  • Spindering
  • Pemindaian aktif
  • Fuzzing
  • Penjalajahan paksa
  • Breaking (intercepting)
  • Mengirim ulang permintaan
Anda dapat menentukan Kebijakan Pindai digunakan untuk mode serangan Pilihan aktif Scan layar.
