Rilis 1.3.0

Berikut perubahan yang dilakukan dalam rilis ini:

Perubahan yang signifikan:

Fuzzing

String dalam tanggapan sekarang dapat fuzzed untuk mencoba menemukan kerentanan.
Token Anti CSRF dapat terdeteksi dan diregenerasikan secara otomatis ketika fuzzing.
Fungsi ini didasarkan pada kode dari OWASP proyek JBroFuzz.

Dinamis sertifikat SSL

Dukungan untuk SSL ditingkatkan dan disederhanakan.
Pengguna sekarang dapat membuat sertifikat sumber mereka sendiri dan mendistribusikan ke klien HTTP.
Lihat Opsi: Sertifikat SSL dinamis untuk lebih jelasnya.

Daemon mode dan API

Memulai ZAP dengan opsi baris perintah "-daemon" akan menyebabkan untuk berjalan di latar belakang 'tanpa' mode, yang berarti bahwa tidak ada UI yang ditampilkan.
API awal ini telah diterapkan dalam XML, JSON dan HTML.
Jika ZAP berjalan sebagai daemon maka API secara otomatis diaktifkan, jika tidak, API harus diaktifkan melalui Opsi API layar.
API dapat navigasikan dengan membuka http://zap/ di browser Anda ketika proxying melalui ZAP.

Beanshell integrasi

The BeanShell interaktif Java shell yang dapat digunakan untuk mengeksekusi skrip BeanShell.
Script ini adalah bentuk sederhana dari Java yang menggunakan banyak unsur dari Java sintaks, tetapi dalam format scripting yang lebih sederhana.
Semua kode Java adalah juga berlaku kode BeanShell.
Integrasi Beanshell di OWASP ZAP memungkinkan anda untuk menulis skrip menggunakan fungsi ZAP dan kumpulan data.
Ini bisa menjadi fitur yang sangat kuat untuk menganalisis aplikasi web.

Internasionalisasi

Menampilkan semua string yang sekarang sepenuhnya diinternasionalkan.

Perubahan kecil:

Hex Tampilan

The Permintaan, Respon tab sekarang menyediakan 'Hex' Melihat opsi yang akan menampilkan permintaan dan tanggapan pokol dalam format hex.

Hasil pencarian

The Tab Pencarian sekarang menampilkan semua contoh dari string yang ditemukan daripada hanya contoh pertama di setiap permintaan atau tanggapan.

Mengecualikan URL

URL dapat secara eksplisit dikecualikan dari scanner aktif, laba-laba dan dari proxy.

Salinan dukungan

Sebagian besar panel sekarang menyediakan pilihan menu 'klik kanan' 'Salinan' termasuk pemindai Port dan panel kekerasan.

Undo/mengulang dukungan

Semua bidang masukan sekarang mendukung Membuka/Mengulangi tindakan menggunakan sistem operasi default Membuka/Mengulangi akselerator:
  • Windows/Linux: Ctrl + Z / Ctrl + Y
  • Mac OS X: Cmd + Z / Cmd + Shift + Z

Port scanner proxy dukungan dan timeout pilihan

Pemindai Port sekarang dapat menggunakan proxy keluar (jika dikonfigurasi) dan batas waktu dalam milidetik sekarang juga dapat diatur.

Tombol istirahat permintaan dan respon

Sekarang ada 2 tombol 'Set Break' untuk memungkinkan istirahat harus ditetapkan pada semua permintaan dan semua tanggapan secara mandiri.

Memperluas situs dan informasi tab tombol

Sekarang ada 2 tombol yang memungkinkan anda untuk beralih antara memiliki Situs dan tab informasi yang diperluas.

Istirahat tab ikon perubahan warna ketika memukul titik istirahat

Sementara tab istirahat ini tidak digunakan ikon adalah salib abu-abu:    .
Ketika break point terkena tab ikon akan berubah ke Palang Merah;    .

Batas waktu disetel

The Opsi: koneksi layar memungkinkan anda untuk mengatur batas waktu dalam hitungan detik untuk memudahkan untuk menguji aplikasi yang lambat.

Update Perpustakaan

Sebagian besar perpustakaan yang digunakan oleh ZAP telah diperbarui ke versi yang terbaru.

Masalah yang diketahui:

Mac OS X: dinamis SSL dan Google Chrome

Saat ini dinamis SSL tidak bekerja ketika menggunakan Google Chrome. Hal ini karena belum terselesaikan masalah dengan Google Chrome dan Mac OS X Keychain. Ketika mengimpor OWASP ZAP Root CA ke dalam keychain dan meminta SSL situs web, pesan galat "sertifikat tidak Valid" yang ditampilkan.
