Rilis 2.4.1

Rilis ini mencakup perbaikan keamanan yang penting - pengguna didesak untuk mengupgrade secepat mungkin.

Salah satu perubahannya adalah kunci API dibuat secara default, artinya setiap aplikasi yang menggunakan API ZAP akan gagal kecuali jika diperbarui menggunakan kunci tersebut.
Kunci API dapat ditemukan di Tampilan pengaturan API
Anda juga dapat mengaturnya melalui perintah menggunakan opsi seperti: 

-config api.key=change-me-9203935709

Detail lebih lanjut lihat https://github.com/zaproxy/zaproxy/wiki/FAQapikey
