モード

ZAP has a 'mode' which can be:

  • Safe - 潜在的に危険な操作を許可しません。
  • Protected - スコープ内のURL上でのみ（潜在的に）危険な動作を行う事が出来ます。
  • Standard - 以前のリリースのように、何でも出来ます。
  • ATTACK - スコープ 上の新しいノードは見つけ次第、 動的スキャンされます。
あなたが意図しているサイトだけを攻撃することを確実にするためにプロテクトモードを使用することをお勧めします。

The mode can be changed via the toolbar (or the ZAP API) and is persisted between sessions.

Examples of the things that will not be possible in either Safe mode or in Protected mode when not acting on URLs in the Scope:
  • Spidering
  • 動的スキャン
  • ファジング
  • 強制ブラウジング
  • Breaking (intercepting)
  • リクエストの再送信
You can define the スキャンポリシー to be used for the Attack mode the オプション 動的スキャン画面.
