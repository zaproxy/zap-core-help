アドオン

アドオンによって、ZAPに機能を追加することができます。
アドオンはZAPの内部機能を完全に利用することによって、非常に強力な新機能を提供することが可能です。
アドオンの管理ダイアログから、オンラインのアドオンマーケットプレースにあるアドオンを動的にインストールすることができます。
通常、ZAP UIへのアドオンの追加と削除は再起動なしに行うことができます。

Add-ons can provide context sensitive help which is dynamically integrated with the help file.
If you are reading this page via a ZAP help file (as opposed to reading it online) then any help pages associated with the add-ons you have installed will be available under here.

アドオンは以下のステータスのどれかが割り当てられます。
     Releasewhich indicates they are of high status and fit for purpose
     Betawhich indicates they are of reasonable status but may be incomplete or need further testing
     Alphaアドオンが開発の初期段階であることを示します。

ZAPでアドオンを利用するための配置

アドオンをZAPで利用できるようにするには、次のいずれかの場所にある必要があります。
  • plugin ディレクトリの配置：ZAPのインストールフォルダ
  • plugin ディレクトリの配置：ZAPのユーザフォルダ
