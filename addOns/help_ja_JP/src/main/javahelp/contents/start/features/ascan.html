アクティブスキャン

アクティブスキャンは、選択されたターゲットに対して既知の攻撃手法を用いることで、潜在的な脆弱性を見つけ出そうとします。

アクティブスキャンはそれらのターゲットへの攻撃です。
所有していないWebアプリケーションに対して使用しないでください。

ZAPトラフィックの識別およびWebアプリケーションファイアウォールの例外設定を容易にするため、ZAPには「AddZapHeader.js」というスクリプトが付属しています。このスクリプトは、ZAPを通過またはZAPから発信されるすべてのトラフィックに特定のヘッダーを追加するために使用できます。 例: X-ZAP-Initiator: 3

アクティブスキャンは特定の種類の脆弱性のみを発見できることに注意してください。
不適切なアクセスコントロールなどの論理的な脆弱性は、アクティブまたは自動化された脆弱性スキャンでは発見されません。
すべての種類の脆弱性を発見するには、アクティブスキャンに加えて常に手動のペネトレーションテストを実施する必要があります。

アクティブスキャンはオプション アクティブスキャン画面を使用して設定されます。
実行されるルールはスキャンポリシーを通じて設定されます。任意の数のスキャンポリシーを持つことができます。
