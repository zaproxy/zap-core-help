ツールメニュー

このメニューは、ツールと一般的なオプションを処理します。

Filter....

This displays the フィルターダイアログ.

Browse API

Opens the ZAP API in the system's default browser.

Encode/Decode/Hash...

This displays the Encode/Decode/Hash dialog.

Manual Request Editor....

This displays the Manual Request Editor dialog.

Options....

This displays the オプションダイアログボックス.

アドオン は他のメニュー項目を追加できます。
