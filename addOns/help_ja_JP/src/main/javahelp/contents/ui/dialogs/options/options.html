オプションダイアログ

オプションダイアログでは、ZAPの設定を行うことができます。
次の画面が含まれます：
    アクティブスキャン
    動的スキャンの入力ベクトル
    アラート
    Anti-CSRFトークン
    API
    ブレークポイント
    更新の確認
    データベース
    表示
    拡張機能
    HTTPセッション
    JVM
    キーボード
    言語
    ルール設定
    スクリプト
    検索
    スパイダー
    統計

初期設定にリセット

「初期設定にリセット」ボタンを押すと、すべてのオプションがZAPの初回起動時の設定値に戻されます。 これには、ZAPが待ち受けるホストやポート、APIキーのリセットも含まれます。
「初期設定にリセット」を行っても、ZAPのルートCA証明書は変更されません。 変更したい場合は、「ネットワーク」→「サーバー証明書」画面から操作できます。
