リリース 2.3.1

本リリースでは以下のような変更が行われました。

  • 問題 81: ZAPがリクエストデータを変更 （ビュー切り替え中)
  • 問題 377: 未実現の依存性が動的スキャンをハングさせる。
  • 問題 1073: 「起動時にロード」としてマークされたスクリプトが削除できない
  • 問題 1114 : core.newSessionがサイトをクリアしない
  • 問題 1155 : 履歴上でリクエストタブの出力形式の切り替えが出来ない
  • 問題 1156 : Proxy gzip decoderがレスポンスヘッダー上のcontent lengthを更新しない
  • 問題 1163 :コマンドライン上でスペースありのホームディレクトリがセット出来ない
  • 問題 1166: zapdb.script の冗長なインデックス
  • 問題 1168 : 「deflate」 content encodingのためのプロキシのサポートを追加
  • 問題 1170 : スパイダーコンテキスト／ユーザ ポップアップメニューが表示されない
  • 問題 1179 : fuzzの結果(Ctrl + click)上で２つのリクエストが選択出来ない
  • 問題 1181 : 脆弱性ページの動的スキャンはひとつしか記録されない
  • Issue 1182 : Alerts of same type for different parameters of same vulnerable page shown only once in "Alerts" tree
  • 問題 1183 : メッセージを削除した後「Alert：タブでノードを選択した時の NullPointerException
  • 問題 1191 : コマンドラインのsessionパラメータに効果がない
  • 問題 1193 : URLパス要素のスキャンは規定でオフにする
  • 問題 1194 : ZAPをdeamonモードで実行した時、コマンドライン引数が拡張機能に渡されない
  • 問題 1196 : AbstractPlugin.bingoが間違って攻撃の証拠を残す
  • 問題 1202 : ロード中のアドオンが正しく初期化されない問題
  • 問題 1203: WordPressの認証スクリプト
  • 問題 1206 : GUIモードのZAPでAPIをとおして新しいセッションを作成する時、「履歴」タブがクリアされない
