Reset to Factory Defaults

The 'Reset to Factory Defaults' button resets all of the options to the values that were set when you first ran ZAP. This includes reeseting the host and port that ZAP is listening on and the API key.
The one configuration item that will not change is the ZAP Root CA Certificate. If you want to change this then you can do that via the Dynamic SSL Certificate screen.

Accessed via

     Top level Tools menu'Options ...' menu item
