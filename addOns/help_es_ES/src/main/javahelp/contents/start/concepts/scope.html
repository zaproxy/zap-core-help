Alcance

El Alcance es el conjunto de URL que está probando, y está definido por los Contextos que ha especificado. <1 /> <1 /> Por defecto, nada está dentro del alcance.

El alcance potencialmente cambia:

  • Qué puedes hacer cuando estás en modo protegido
  • Qué se muestra en la pestaña Historial
  • Protegido: el usuario solo puede realizar acciones (potencialmente) peligrosas en las URL del ámbito.
  • Estándar: como en versiones anteriores, el usuario puede hacer cualquier cosa
  • ATTACK: los nuevos nodos que están en Scope se escanean activamente tan pronto como se descubren
Se recomienda que defina un nuevo Contexto para cada aplicación web que conforma el sistema que está probando, y establecerlos en el alcance mientras prueba cada uno.
