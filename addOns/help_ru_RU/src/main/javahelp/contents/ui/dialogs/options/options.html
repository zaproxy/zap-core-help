Диалоговое окно параметров

Диалоговое окно «Параметры» позволяет настроить ZAP.
Он включает в себя следующие экраны:
    Активное Сканирование
    Входные векторы активного сканирования
    Оповещения
    Анти-CSRF токены
    API
    Прерывания
    Check for Updates
    Database
    Display
    Extensions
    Global Exclude URL
    HTTP Sessions
    JVM
    Keyboard
    Language
    Passive Scan Rules
    Passive Scan Tags
    Passive Scanner
    Rule Configuration
    Scripts
    Search
    Spider
    Statistics

Сброс до заводских настроек по умолчанию

Кнопка «Восстановить заводские настройки» сбрасывает все параметры до значений, которые были установлены при первом запуске ZAP. Это включает в себя сброс хоста и порта, которые прослушивает ZAP, и ключа API.
The one configuration item that will not change is the ZAP Root CA Certificate. If you want to change this then you can do that via the Network / Server Certificates screen.
