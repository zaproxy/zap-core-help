This allows you to enable and disable the rules that are run when performing an active scan.
The first screen allows you to define the default levels as well as the levels for all of the rules in a specific category.

The category screens allow you to define the levels for every individual rule.

Locked

Locked policies do not allow the use of any other scan rules than the ones defined in their configuration, preventing the inadvertent use of other/new scan rules.

Порог

Это контролирует, насколько вероятно, что ZAP сообщит о потенциальных уязвимостях.
  • If you select Off then the scan rule won't run.
  • Если вы выберете «Низкий», возникнет больше потенциальных проблем, которые могут увеличить количество ложных срабатываний.
  • Если вы выберете «Высокий», возникнет меньше потенциальных проблем, что может означать, что некоторые реальные проблемы будут упущены (ложноотрицательные результаты).

Сила

Это контролирует количество атак, которые будет выполнять ZAP.
Если вы выберете «Низкий», будет использоваться меньше атак, что будет быстрее, но при этом могут быть пропущены некоторые проблемы.
Если вы выберете «Высокий», будет использоваться больше атак, которые могут выявить больше проблем, но займут больше времени.
Уровень Insane обычно следует использовать только для небольших частей приложения, так как это может привести к очень большому количеству ошибок. используются атаки, которые могут занять значительное время.

Примечание: Имейте в виду, что использование прозвища "Безумие" в отношении сканирования сила - это просто имя, выбранное для обозначения максимальной силы сканирования, это не комментарий или ссылка на психическое здоровье или личную стабильность.
