الوظائف الإضافية

إضافات إضافة وظائف إضافية انطلق.
لديهم حق الوصول الكامل إلى جميع انطلق الداخلية ، لذلك يمكن أن توفر قوية جدا من الميزات الجديدة.
يمكنك حيوي تثبيت إضافات من على الانترنت إضافة على السوق عن طريق "إدارة الوظائف الإضافية" الحوار.
يمكنك عادة إضافة وإزالة الإضافات إلى من انطلق واجهة المستخدم دون الحاجة إلى إعادة تشغيل.

Add-ons can provide context sensitive help which is dynamically integrated with the help file.
If you are reading this page via a ZAP help file (as opposed to reading it online) then any help pages associated with the add-ons you have installed will be available under here.
