API

ZAP, ZAP ile programlı olarak etkileşime girmenize izin veren bir Uygulama Programlama Arayüzü (API) sağlar.

API, JSON, HTML ve XML formatlarında bulunur. <0 /> API'yı keşfetmenizi ve kullanmanızı sağlayan basit bir web kullanıcı arayüzü, http: // zap / URL'sinde mevcuttur. ZAP vasıtasıyla veya ZAP tarafından dinlenen ev sahibi ve port yoluyla, örneğin http: // localhost: 8080 / .

Varsayılan olarak yalnızca ZAP çalışan makine API erişebilir. ZAP'ı proxy olarak kullanabilen diğer makinelere API'ye erişim izni verebilirsiniz. API, Seçenekler API ekranı kullanılarak yapılandırılmıştır.

The API provides access to most of the core ZAP features such as the active scanner.
Future versions of ZAP will increase the functionality available via the APi.
