Options dialog

The Options dialog allows you to configure ZAP.
It include the following screens:
     Connection
     Local Proxy
     Authentication
     Certificate
     Display
     Check for Updates
     Spider
     Active Scan
     Passive Scan
     Brute Force
     Port Scan

Accessed via

     Top level Tools menu'Options ...' menu item
