Scan Pasif

ZAP secara bawaan melakukan pemindaian secara pasif pada pesan HTTP (permintaan dan respon) dikirim ke aplikasi web yang dilakukan tes.
Pemindaian pasif tidak merubah permintaan atau respon dalam cara apapun dan karena itu aman untuk digunakan.
Peninjauan dilakukan di thread latar belakang untuk memastikan hal tersebut tidak memperlambat eksplorasi dari aplikasi.

Perilaku (utama) dari peninjauan pasif dapat dikonfigurasi menggunakan Option Passive Scanner Screen.

Peninjauan pasif dapat juga digunakan untuk menambahkan secara otomatis tags dan pemeliharaan alerts untuk masalah potensial.
Seperangkat aturan untuk memberikan label secara otomatis disediakan secara bawaan. Hal ini dapat diubah, dihapus atau ditambahkan melalui Options Passive Scan Tags screen.

Peringatan yang diajukan oleh pemindaian pasif dapat dikonfigurasi menggunakan Option Passive Scan Rules screen.
