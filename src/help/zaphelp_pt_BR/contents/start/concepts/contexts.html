Contextos

Contextos são uma maneira de relacionar um conjunto de URLs.
Você pode definir quaisquer contextos que queira, mas espera-se que um contexto corresponder a um aplicativo web.
É recomendável que você defina um novo contexto para cada aplicativo web, cada sistema que você esteja testando, e os configure no escopo ao testar cada um.

Contextos são definidos como um conjunto de expressões regulares (regexs), que são aplicadas a todas as URLs na Aba de Sites.
Você pode configurar contextos através de:

  • Itens de menu de clique direito nas abas Sites ou Histórico
  • Diálogos de contextos de sessão.
