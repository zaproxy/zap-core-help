Guia do Usuário OWASP ZAP

Welcome to the The OWASP Zed Attack Proxy (ZAP) User Guide.

This is available both as context sensitive help within ZAP and online at https://github.com/zaproxy/zap-core-help/wiki/HelpIntro

O ZAP é uma ferramenta de teste de penetração de fácil uso que visa identificar vulnerabilidades em aplicações web.

Ele é projetado para ser usado por pessoas com uma vasta gama de experiência em segurança e assim é ideal para desenvolvedores e testadores de funcionalidades que são novatos em testes de penetração.

O ZAP oferece scanners automatizados, bem como um conjunto de ferramentas que permitem a você encontrar vulnerabilidades de segurança manualmente.

Se você é novo no ZAP recomendamos que você dê uma olhada na seção Iniciando.

ZAP é um braço da variante de código aberto do Paros Proxy.
