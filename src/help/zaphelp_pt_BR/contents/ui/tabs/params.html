Aba de parâmetros

Isto mostra um resumo dos parâmetros de que um site usa.

Sites podem ser selecionados através da barra de ferramentas ou pela Aba de Sites.
Em cada parâmetro, você pode ver:
     O tipo - Cookie, formulário ou URL
     O nome do parâmetro
     O número de vezes que foi usado
     O número de valores exclusivos/únicos
     A porcentagem de mudanças, onde 0 significa que apenas um valor foi usado e 100 significa que todos os valores são únicos
     As flags - incluindo sinalizadores de cookie, anticsrf e sessão
     Alguns dos valores - o conjunto completo de valores pode não ser visível

Menu do botão direito

Clicar com o botão direito em um nó faz aparecer um menu que lhe permitirá:

Localizar

Isto irá mostrar todos os exemplos do parâmetro selecionado na Aba de Busca.

Sinalizar como token Anti CSRF

Isto sinalizará o parâmetro como um token Anti CSRF.

Remover o sinalizador como token Anti CSRF

Isto irá remover o sinalizador de token Anti CSRF do parâmetro.

Sinalizar como token de sessão

Isto marcará o parâmetro como um token de sessão para o Site atual e irá notificar a ferramenta de HTTP Sessions nesse sentido.

Remover o sinalizador de token de sessão

Isto marcará o parâmetro como um token de sessão para o Site atual e irá notificar a ferramenta de HTTP Sessions nesse sentido.
