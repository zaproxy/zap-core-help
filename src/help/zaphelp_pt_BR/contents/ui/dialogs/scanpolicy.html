Scan Policy Dialog

Isso permite a você ativar e desativar as regras que são executadas quando fizer uma varredura ativa.
A primeira tela permite que você defina os níveis default, bem como os níveis de todas as regras em uma categoria específica.

As telas de categoria permitem definir os níveis para cada regra individualmente.


Note que regras de varredura passiva não são mais gerenciadas através desta caixa de diálogo, mas em vez disso são gerenciadas por meio das Opções de regras de varredura passiva.

Limiar

Essa opção controla como o ZAP informará vulnerabilidades potenciais em relatório.
  • If you select Off then the scanner won't run.
  • If you select Low then more potential issues will be raised which may increase the number of false positives.
  • If you select High then fewer potential issues will be raised which may mean that some real issues are missed (false negatives).

Força

Essa opção controla o número de ataques que o ZAP executará.
Se você selecionar baixo então menos ataques serão usados, o que será rápido, mas pode ignorar alguns problemas.
Se você selecionar alto então mais ataques serão usados, o que pode localizar mais problemas, mas isso vai demorar mais.
O nível "Insano" normalmente deve ser usado apenas para pequenas partes de um aplicativo, porque pode resultar em um grande número de ataques, o que pode levar um tempo considerável.
