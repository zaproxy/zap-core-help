Release 1.4.0

Este release trouxe as seguintes mudanças:

Alterações significativas:

Problema 133: Fazer realce de sintaxe no painel de resposta

Os painéis HTML agora oferecem realce alternável da sintaxe.

Problema 153: integração fuzzdb

The fuzzer includes fuzzdb (https://github.com/fuzzdb-project/fuzzdb) fuzzing files.
Note que alguns arquivos de fuzzdb foram ignorados porque os scanners de vírus indicam que eles contêm vírus.
Você pode substituí-los (e atualizar o fuzzdb) baixando a versão mais recente do fuzzdb para a biblioteca 'fuzzers'.

Problema 212: Análise de parâmetro

Uma nova aba Params mostra um resumo de todos os parâmetros que um site usou.

Problema 228: varredura de XSS aperfeiçoada

O scanner ativo de Cross Site Scripting foi reescrito do zero para encontrar mais problemas potenciais de XSS e apontar menos falsos positivos.

Problema 244: transferir as checagens passivas do Watcher

As seguintes verificações foram transferidas do Watcher (graças a Chris Weber por resolver isso):
    Check.Pasv.CrossDomain.ScriptReference.cschecar a inclusão de arquivos javascript entre domínios.
    Check.Pasv.Header.CacheControl.csverifica o cabeçalho de cache-control HTTP nas páginas SSL.
    Check.Pasv.Header.ContentTypeMissing.csverifica-se se o cabeçalho HTTP Content-Type está presente.
    Check.Pasv.Header.FrameOptions.csverifica-se que o X-FRAME-OPTIONS não está ausente ou setado de modo inseguro.
    Check.Pasv.Header.IeXssProtection.csverifica se o X-XSS-Protection não foi ajustado para desativar a protecção de XSS do IE.
    Check.Pasv.Header.MimeSniff.cscheca se o X-CONTENT-TYPE-OPTIONS foi definido.
    Check.Pasv.InformationDisclosure.DatabaseErrors.csverifica se há mensagens de erro de banco de dados.
    Check.Pasv.InformationDisclosure.DebugErrors.csverifica se há mensagens de erro de depuração.
    Check.Pasv.InformationDisclosure.InUrl.csverifica se há vazamento de informações nos parâmetros da URL.
    Check.Pasv.InformationDisclosure.ReferrerLeak.csverifica se há vazamento de informações no cabeçalho HTTP Referer.

Problema 253: Extensões Plugáveis

Extensões inteira podem agora ser plugadas no ZAP dinamicamente, com acesso total a todos os recursos do ZAP.
