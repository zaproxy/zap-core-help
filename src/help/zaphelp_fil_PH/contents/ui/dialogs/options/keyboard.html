Mga Opsyon ng Keyboard screen

Ang screen na ito ay nagpapahintulot sa iyo na magkumpigura ng mga keyboard shortcut para sa lahat ng mga menu ng ZAP.

Ang pag-click sa alinman sa mga hilera ng talahanayan ay magdadala ng kahon ng diyalogo na nagpapahintulot sa iyo na baguhin ang mga keyboard shortcut.

'Aksyon' Cheatsheet

Ang pag-click sa button na ito ay makagagawa ng isang pahina ng HTML na magpapakita sa lahat ng mga keyboard shortcut na pinagsunod-sunod sa 'aksyon'.
Ang button na ito ay pinagana lamang kung ang API ay pinagana at ang iyong operating systems ay sumusuporta ng pagbubukas ng mga URL sa browser.

'Key' Cheatsheet

Ang pag-click sa button na ito ay makakagawa ng isang pahina ng HTML na magpapakita sa iyo sa lahat ng mga keyboard shortcut na pinagsunod-sunod sa 'key'.
Ang button na ito ay pinagana lamang kapag ang API ay pinagana at ang iyong operating system ay sumusuporta ng pagbubukas ng mga URL sa browser.

I-reset

Ang pag-click sa button na ito ay magri-reset sa lahat ng mga shortcut pabalik sa mga default nila.
