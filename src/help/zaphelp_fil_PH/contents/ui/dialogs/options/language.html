Screen ng wika ng mga pagpipilian

Ang mga screen ng wika ay nagpapahintulot sa iyo na magkumpigura ng:

Pagpipilian ng Wika

Ang wika ay gumagamit ng ZAP para display nito.
Isang set ng mga wika ay binuo sa ZAP, subalit mas maraming wika ay maaaring magagamit matapos ang huling release.
Lagyan ng tsek ang ZAP download page para sa mga pinakabagong language pack makukuha.

Gamitin ang Lunan ng sistema para sa pagformat

Kung ang pagformat ng numero, petsa, at oras ay tapos na sa Lunan ng sistema sa halip na ang Lunan ng napiling wika.

Mag-import ng pakete ng wika

Bukod sa pagpili ng wika na ginagamit para sa mga displey ng ZAP, ang pag-import tool ay nagbibigay-daan sa iyo upang i-import ang mga bagong pakete ng wika o wikang file update. Laging magkakaroon ng isang pakete ng wika file extension zaplang. (Hal. wika-pack-en_GB.zaplang)
Matapos ang pag-import mo agad magagawang upang pumili ang bagong import na wika mula sa wikang pagpipilian drop down menu sa tuktok ng screen.
Tandaan na kailangan mong i-restart ang ZAP upang sa wakas mag-apply ng bagong wika.
